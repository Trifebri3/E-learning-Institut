📖 Tentang Proyek
LMS Institut Hijau Indonesia adalah platform manajemen pembelajaran (Learning Management System) yang dirancang khusus untuk mendukung program edukasi Institut Hijau Indonesia. Sebagai NGO yang bergerak di bidang lingkungan dan sosial, platform ini menjadi jembatan bagi para pemuda untuk mendapatkan literasi ekologi dan pengembangan diri secara digital.

🎯 Tujuan Program
Literasi Lingkungan: Menyediakan modul pembelajaran mengenai isu krisis iklim dan pelestarian alam.

Pemberdayaan Pemuda: Mengasah keterampilan kepemimpinan sosial berbasis keberlanjutan.

Aksesibilitas: Memastikan program pendidikan dapat dijangkau oleh relawan dan pemuda di seluruh penjuru Indonesia.

✨ Fitur Utama
🌱 Curriculum Green-Focused: Manajemen kursus yang terstruktur untuk topik-topik lingkungan.

👥 Management User (Roles):

Admin: Pengelolaan konten dan statistik program.

Mentor/Instruktur: Memberikan penilaian dan bimbingan edukasi.

Peserta (Pemuda): Akses materi, kuis, dan sertifikat digital.

📊 Progress Tracking: Memantau perkembangan belajar setiap individu secara real-time.

📜 Digital Certificate: Pemberian sertifikat otomatis setelah menyelesaikan modul pembelajaran.

📱 Responsive Design: Optimal diakses melalui smartphone untuk mendukung mobilitas aktivis di lapangan.

🛠️ Stack Teknologi
Proyek ini dibangun menggunakan teknologi mutakhir untuk memastikan performa yang stabil:

Core Framework: Laravel (Elegant PHP Framework)

Frontend: Blade Templating / Tailwind CSS (Modern & Clean Design)

Database: MySQL / PostgreSQL

Server: Nginx / Apache

⚙️ Cara Instalasi
Ikuti langkah-langkah berikut untuk menjalankan project di lingkungan lokal:

Clone Repository

Bash
git clone https://github.com/username/e-learning-ih-indonesia.git
Instalasi Dependensi

Bash
composer install
npm install && npm run dev
Konfigurasi Database

Salin file .env.example menjadi .env

Atur koneksi database di dalam file .env

Migrasi & Generate Key

Bash
php artisan key:generate
php artisan migrate --seed
Akses Platform

Bash
php artisan serve
Buka http://localhost:8000 di browser Anda.
